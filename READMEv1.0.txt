Installation
	Run the following command in the project directory python -m SimpleHTTPServer
	For Python3.4 on windows 8 there is a file provided to launch the server called 'python'
	Once the server is running, navigate to localhost:8000
	You should see the login screen

Operation
	Login is currently blank for easy access to the program
	There is a skeleton where each widget is, or will be
	Basic functionality is in place for each component
